1、先创建一个远程仓库
2、创建完后，在本地需要提交的项目，点击右键，再点击Git bash here
	1.创建一个文件：
		echo "# selfcode" >> README.md
	2.创建一个本地仓库：
		git init
	3.添加一个或者多个文件
		git add [XX][XX]   .是选择所有
	4.添加注释
		git commit -m "first commit"
	5.帮本地仓库默认指向一个远程仓库
		git remote add origin https://github.com/zhangzor/selfcode.git
	6.提交代码
		git push -u origin master

更新git文件，代码：
	1、git add [文件名]
	2、git commit -m "注释"
	3、git push
	
修改内容:
	1、git add .
	2、git commit -m "注释"
	3、git push

1.使用git clone命令将新建的git 项目信息下载到本地
git pull 也是拉取代码 / git pull origin 分支名
git clone  git address
2.进入git项目的本地文件夹，将自己的项目复制到这个git 文件夹，查看状态，提交所有的新文件
git status 
git add -A
git commit -a -m"first commit"
3.上传更新到git 版本库
git push

注：git add上传本地项目所有变化的命令三种有 git add -A、git add -u、git add .
git add -A  提交所有变化
git add -u  提交被修改(modified)和被删除(deleted)文件，不包括新文件(new)
git add .  提交新文件(new)和被修改(modified)文件，不包括被删除(deleted)文件
