1、先创建一个远程仓库
2、创建完后，在本地需要提交的项目，点击右键，再点击Git bash here
	1.创建一个文件：
		echo "# selfcode" >> README.md
	2.创建一个本地仓库：
		git init
	3.添加一个或者多个文件
		git add [XX][XX]   .是选择所有
	4.添加注释
		git commit -m "first commit"
	5.帮本地仓库默认指向一个远程仓库
		git remote add origin https://github.com/zhangzor/selfcode.git
	6.提交代码
		git push -u origin main
